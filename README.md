# プロジェクト概要

このリポジトリには様々なウェブアプリケーションプロジェクトが含まれています。
これらのプロジェクトは2021年頃に作成されました。
以下に各プロジェクトの概要を示します。

## 英単語マネージャー (`english/`)
英単語とその日本語訳を登録・管理するためのアプリケーションです。

### 機能
- 英単語の追加
- 英単語リストの表示
- 登録情報の変更・削除
- 英単語の検索

### データベース
`db_english` データベースを使用します。詳細は `db.sql` を参照してください。

---

## Lime - チャットアプリケーション (`lime/`)
ユーザー登録を行い、友達とリアルタイム（風）にチャットができるアプリケーションです。

### 機能
- ユーザー登録（新規登録）
- ログイン
- 友達との個別チャット機能
- メッセージの送信・受信履歴表示

### データベース
`db_lime` データベースを使用します。詳細は `db.sql` を参照してください。

---

## 会員管理システム (`membership/`)
会員登録、ログイン、会員情報の管理を行うシステムです。

### 機能
- 新規会員登録（名前、年齢、性別、メール、パスワード）
- ログイン・ログアウト
- アカウント情報の変更
- 退会処理（ステータス変更）
- 会員リスト表示（管理者向け）

### データベース
`db_member` データベースを使用します。詳細は `db.sql` を参照してください。

---

## Mytube - 動画共有プラットフォーム (`youtube/`)
動画のアップロード、視聴、コメント投稿ができる動画共有プラットフォームのプロトタイプです。

### 機能
- ユーザー登録・ログイン
- 動画のアップロード（サムネイル画像と動画ファイル）
- 動画の視聴
- 動画へのコメント投稿
- 投稿者本人による動画の削除機能

### データベース
`db_youtube` データベースを使用します。詳細は `db.sql` を参照してください。
